# Implement Subnet Detail Page

Implements the `/subnets/[id]` page for web_15_autostats showing detailed subnet information with candlestick chart and trading interface.

## What's included

- Subnet detail page with two-column layout (380px sidebar + content area)
- Left sidebar with comprehensive statistics (price, financial data, token stats, trading stats, subnet data)
- Candlestick chart with 3 candle sizes (1H, 4H, 1D) using lightweight-charts v5
- Place Order section with Buy/Sell forms
- Transaction History table
- Back button navigation to subnets list

**New files:**

- `src/components/pages/SubnetDetailPageContent.tsx`
